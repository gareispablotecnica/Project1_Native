# Proyecto 1 - React Native

Aplicación desarrollada con **React Native** y **Expo** que muestra un catálogo de productos utilizando tarjetas con imágenes, nombre, precio y un botón para agregar al carrito.
- Curso: 7°3
- Grupo de Taller: 7.6
- Docente: Gareis Pablo

##  Características

- Fondo personalizado utilizando `ImageBackground`.
- Tarjetas de productos.
- Imagen de cada producto.
- Nombre del producto.
- Precio.
- Botón "Agregar al Carrito".
- Estilos realizados con `StyleSheet`.

## ▶️ Instalación

1. Clonar el repositorio.

```bash
git clone https://github.com/usuario/proyecto1_native.git
```

2. Entrar en la carpeta.

```bash
cd proyecto1_native
```

3. Instalar las dependencias.

```bash
npm install
```

4. Ejecutar el proyecto.

```bash
npx expo start
```

Luego podés abrir la aplicación desde:

- Expo Go (Android/iOS)
- Emulador Android
- Simulador iOS

##  Funcionalidades

- Visualización de productos mediante tarjetas.

- Botón para agregar productos al carrito (muestra una alerta).
